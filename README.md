# syougi.js

将棋をJavaScript + jQueryで実装したブラウザゲームです。

*2011年、22歳のときに作ったものです。レガシーなコードはご容赦ください 🙏*

## 🚀 Play

**https://oshiro-kazuma.github.io/syougi.js/**

<img src="v1/screen_shot.png" width="420" alt="syougi.js screenshot">

## 🛠 Tech Stack

- JavaScript / jQuery 1.6
- jQuery UI 1.8

## 💻 Dev

```bash
make start
```

ブラウザで http://localhost:8080 が開きます。
